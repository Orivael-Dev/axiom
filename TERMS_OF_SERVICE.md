## 2. Proprietary Trade Secret — Two-Layer Evaluation Pattern

**The Two-Layer Evaluation Pattern — including Teacher-Student honesty evaluation,
Kid-Parent factual accuracy scoring, and Doctor-Patient source verification —
constitutes proprietary trade secret intellectual property of Orivael Inc.
Use of the pattern in competing governance frameworks without license is prohibited.**

The Two-Layer Evaluation Pattern encompasses:

- The architectural pattern in which a primary agent generates outputs that are
  systematically evaluated by a secondary verification agent before scoring,
  certification, or delivery.

- **Teacher-Student honesty evaluation**: A teacher agent evaluates student agent
  responses for benchmark honesty across demographic variants, producing a ledger
  of honest/dishonest/biased verdicts used for constitutional certification.

- **Kid-Parent factual accuracy scoring**: A parent verification agent cross-checks
  child agent outputs against known factual ground truth, flagging fabrications and
  scoring accuracy for regulatory compliance evidence.

- **Doctor-Patient source verification**: A doctor verification agent applies a
  five-tier evidence registry to patient retrieval agent synthesis, approving,
  flagging, or constitutionally blocking medical claims before user delivery.

- The specific evaluation runner, ledger format, demographic variant generation
  methodology, and certification pipeline that implements the above patterns.
