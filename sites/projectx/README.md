# AXIOM // ORIVAEL Dashboard

A runnable React/Vite version of the AXIOM dashboard component.

## Run locally

```bash
npm install
npm run dev
```

Then open the local Vite URL in your browser.

## Build for production

```bash
npm run build
npm run preview
```

## Files

- `src/App.jsx` contains the AXIOM dashboard UI.
- `src/main.jsx` mounts the React app.
- `src/index.css` provides basic full-screen layout reset.
