KNOWN GAPS (surfaced by Chaos Suite):

[x] GAP-01: MUTATES/CANNOT_MUTATE overlap not flagged by validator
    Category: Contradiction (C2) -- FIXED 2026-04-15
    Detail: Added Phase 1 syntax check in validate() -- fields in both
            MUTATES and CANNOT_MUTATE now return status=invalid with
            "constitutional conflict" error message.

[x] GAP-02: CANNOT_MUTATE not enforced at save_axiom layer
    Category: Adversarial Override (O2) -- FIXED 2026-04-15
    Detail: save_axiom() now loads the original file on disk, compares
            each CANNOT_MUTATE field, and raises AxiomConstitutionalViolation
            if any protected field was changed. New exception class added
            to axiom_files/parser.py.

[x] WHEN construct -- declarative conditional concept activation
    FIXED 2026-04-15
    Detail: New WHEN block in .axiom files. Rules follow pattern:
              "if task involves X, activate ConceptName"
            Parser: compile_decision_table(), apply_decision_table(),
              get_prompt_with_when() added to parser.py
            Validator: Phase 3d -- entries must contain "activate"
            Concepts: AmbiguityResolution added to concepts.axiom
            worker.axiom: 3-rule WHEN block added (UncertaintyBound,
              RewardGuard, AmbiguityResolution)

[x] DELEGATES construct -- declarative agent routing
    FIXED 2026-04-16
    Detail: New DELEGATES block in .axiom files. Rules follow pattern:
              "Source -> Target (on: trigger)"
            Parser: compile_delegates(), get_delegates_for() added
              to parser.py; serialization in save_axiom()
            Validator: Phase 3e -- entries must contain "->"
            Concepts: RecoveryMode added to concepts.axiom
            worker.axiom: 2-rule DELEGATES block added
              (Worker->Rewriter on RecoveryMode,
               Worker->Evaluator on output_ready)
