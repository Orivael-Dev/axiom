3. Two gate checks of the SAME benign packet

A packet whose payload would normally pass the intent classifier (a benign request) is gated by the bonded-pair authority register. Between the two checks, only one thing changes: the ledger entry for this pair. The primary token's bytes and signature are identical in both checks.

The primary token's signature did not change. No key was rotated. No consumer was notified. The next gate request was denied because the gate consults the bonded-pair register, and the register has the new state.

4. What this is not

This primitive is not quantum entanglement. There are no Bell correlations and no non-local effects. The name in the implementation is “bonded paired-token with atomic state register.” The useful property is: authority can be revoked across an arbitrary distance with a single register-flip, without rotating the keys that signed the original tokens. That property is what the demonstration above is verifying byte-for-byte.
